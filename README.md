## Installation

#### Require
- NodeJS - コア処理
- MySQL (or MariaDB) - 記憶保存
> SQLデータは公開していません

```
git clone https://github.com/BotGirls/AkariBot-Core.git
cd AkariBot-Core
cp config.sample.js config.js
vi config.js # トークンなど

npm install
node main.js
```
